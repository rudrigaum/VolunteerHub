# Security Policy

## Supported Versions

Currently, as the project is in active initial development, only the latest `main` branch is supported with security updates. 

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Security is a top priority for VolunteerHub. We greatly appreciate the efforts of the security community and researchers who help us identify vulnerabilities.

If you discover a security vulnerability within this project, please **DO NOT** open a public issue. 

Instead, please send an email directly to the repository owner or use the GitHub Security Advisory feature to report it privately. 

We will acknowledge receipt of your vulnerability report as soon as possible, and strive to send you regular updates about our progress. Once the issue is resolved, we will publish a security advisory acknowledging your contribution (if you desire).
